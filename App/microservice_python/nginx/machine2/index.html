Machine 2

Product catalog online, inventaire accessible.

Front applicatif du microservice produit. Cette machine sert la consultation et l'ajout d'articles dans le catalogue via le proxy Nginx expose en port 8082.

Microservice Product Catalog
Endpoint /api/product
Methodes GET POST
Donnees Liste de produits
HTTP 200

Catalogue disponible

Le frontal redirige correctement vers l'API Flask pour lister les produits et enregistrer de nouvelles references.

Exemple de requete

GET http://localhost:8082/api/product pour lire le catalogue expose par le microservice.

Exemple de payload

{"id":"P-100","name":"Laptop","price":1299} a envoyer en POST pour ajouter un produit.
