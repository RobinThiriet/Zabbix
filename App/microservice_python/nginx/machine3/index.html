Machine 3

Order workflow online, commandes disponibles.

Front applicatif du microservice commande. Cette machine permet de consulter et creer des commandes via le proxy Nginx publie sur le port 8083.

Microservice Order Management
Endpoint /api/order
Methodes GET POST
Donnees Liste de commandes
Service web disponible

Gestion des commandes

Le frontal Nginx expose correctement l'API Flask pour le suivi et la creation des commandes metier.

Exemple de requete

GET http://localhost:8083/api/order pour lister les commandes actuellement en memoire.

Exemple de payload

{"id":"O-100","user_id":"U-1","product_id":"P-100"} a envoyer en POST pour creer une commande.
